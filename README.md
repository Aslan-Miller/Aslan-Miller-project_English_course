# Chatty English Bot - Бот для изучения английского языка

Telegram-бот с системой платных подписок для доступа к материалам по изучению английского языка.

## Функционал

### Для пользователей:
- **Бесплатный доступ**: Открытая группа с базовыми материалами
- **Платные курсы**: Закрытая группа с расширенными материалами
- **Система подписок**: Автоматическая проверка срока действия подписки
- **Простая оплата**: Интеграция с Kaspi для оплаты подписки

### Для администратора:
- **Админ-панель**: Управление через Telegram
- **Статистика**: Просмотр количества пользователей и активных подписок
- **Управление платежами**: Подтверждение и отклонение платежей
- **Список пользователей**: Просмотр всех зарегистрированных пользователей
